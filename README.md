# Systems-Project-Spring-2023


## Inventory Management System


### About the Project
This project is the second version of the Inventory Management project built spring of 2023 for our college.  This application is to help keep better records of the ITC checkout system.  Students in the IT programs can check out different kits and items based on their individual needs.  This application allows the user to check in/out kits and items to students and records the transactions and information within a database.
